# Help Center screenshots

Drop the PNG files below into this folder (`public/help/`). They power the
"About this Page" illustrated guides in the Help Center. Until a file is present,
that guide shows a neutral placeholder and the numbered description list still
works — so the app is fully usable before and after you add images.

Recommended: capture at **1280×800** (or 2× / Retina for crispness), light OR dark
theme (either is fine — pick one and stay consistent), and crop to the page's
content area (exclude the OS chrome). Keep the numbered regions in the app's
Help → About list visible in the shot.
